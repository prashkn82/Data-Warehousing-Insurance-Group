Premium Trend Dashboard
1. Dashboard Purpose
The Premium Trend Dashboard provides a high‑level, visual summary of premium collection patterns across months, quarters, years, regions, agents, and policy types.
It is designed for senior management, underwriting teams, and finance departments to quickly understand revenue trends and make strategic decisions.

This dashboard is one of the most visually rich Business Objects assets in the Data Warehousing Insurance Group project.

2. Data Sources Used
Policy_Fact  
Premium amounts, coverage, policy duration.

Policy_Dim  
Policy type, status, attributes.

Agent_Dim  
Agent region and performance.

Customer_Dim  
Customer demographics.

Date_Dim  
Calendar hierarchy for premium trend visualization.

3. Key Dashboard Components
A. Monthly Premium Trend Chart
Shows:

Premium collected each month

Seasonal variations

Month‑over‑month growth

B. Quarterly Premium Summary
Displays:

Total premium per quarter

Quarter‑over‑quarter comparison

Trend indicators (upward/downward arrows)

C. Year‑Over‑Year Premium Comparison
Includes:

Premium totals for each year

Growth percentage

Historical premium trend line

D. Region‑Wise Premium Contribution
Shows:

Premium generated by each region

Regional ranking

Heat map visualization

E. Policy Type Premium Distribution
Displays:

Premium contribution by policy type

Life vs Auto vs Health vs Property

Pie chart or bar chart representation

F. Agent Premium Contribution
Includes:

Premium generated per agent

Top 10 agents

Underperforming agents

4. BO Features Used in This Dashboard
Charts & graphs (line, bar, pie, heat map)

Drill‑down from year → quarter → month

Prompts for filtering by:

Region

Policy type

Agent

Date range

Conditional formatting for:

High‑premium months

Low‑premium regions

Sudden drops in premium collection

Trend indicators (arrows, color coding)

5. Sample BO Query Logic (Semantic Layer)
Policy → Fact Join
Code

Date Dimension Join
Code

Agent Join

Support pricing and underwriting strategy
