📘 Project Overview

This academic project focuses on designing and implementing a Data Warehouse for an Insurance Group, enabling unified storage, integration, maintenance, and analysis of heterogeneous insurance data.

The solution was built using Oracle 9i, Informatica PowerCenter, and Business Objects, adhering to industry-standard ETL and dimensional modelling practices.

The project ensures:

Centralised data storage

Historical data maintenance

Faster reporting and analytics

Improved decision‑making for insurance operations

🎯 Objectives

Integrate heterogeneous insurance data into a single centralised warehouse.

Implement Slowly Changing Dimensions (SCD) – Type 1, Type 2, Type 3.

Build Fact & Dimension tables for analytical processing.

Develop ETL workflows using Informatica PowerCenter.

Generate business reports using Business Objects.

Maintain database integrity and performance using Oracle DBA best practices.
